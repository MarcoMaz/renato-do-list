# Project React Template
### Description

Work-in-progress template to have a ready to use react project.

### Installation

1. Clone the repo
2. `npm install`
3. `npm run start`

### Included:
* React
* Redux
* Typescript
* Sass
* React Router
* Jest for Testing
* Linters (Eslint and Prettier)
* Saved on Github Pages
* PWA enabled
